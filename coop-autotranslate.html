Granblue Co-op English Auto-Translation Userscript

EXPERIMENTAL

Automatically translates co-op room descriptions into English, using translations based on /u/finis_caelorum's Co-op Lobby Phrasebook. Translated text appears in green.

This script only "translates" by simple text replacement, so be especially careful when untranslated text remains! The untranslated text could significantly change the meaning of the rest of the message.

Right-clicking on a translated description will open a text field that you can copy from into a better translation website, or send to a friend for help.

Installation

  1. Install the Tampermonkey browser extension
  2. then, Install this userscript
